GradeCalc AI — assets folder
=============================

This app is fully self-contained and works without adding anything here.
A default favicon.svg is already included.

Optional additions if you want to customize:
  - favicon.svg    -> replace to change the browser tab icon (32x32 or SVG, any size)
  - og-image.png    -> add a 1200x630 image if you want a social-share preview card
                       (then add <meta property="og:image" content="assets/og-image.png">
                       in the <head> of index.html)

Fonts (Fraunces, Inter, IBM Plex Mono) and the PDF/Image export libraries
(jsPDF, html2canvas) are loaded from public CDNs in index.html — no local
files are required for those either.
